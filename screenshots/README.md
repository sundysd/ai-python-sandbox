# Screenshots Directory

This directory contains screenshots of the AI Python Sandbox application.

## How to Add Screenshots

1. Run the application: `streamlit run app.py`
2. Open your browser to `http://localhost:8501`
3. Take screenshots of different views:
   - Main interface with input field
   - Code generation examples
   - Execution results
   - History/logs section
4. Save images as PNG files in this directory
5. Update the README.md with the correct image paths
